Консольная утилита обработки заказов
На входе - имена файлов формата csv, через пробел
В качестве аргумента через ключ -о указывается имя выходного файла
Формат CSV:
1 колонка - товар/материал (строка без спец.символов, только английские буквы и цыфры)
2 колонка - количество товара (целочисленное, неотрицательное)

Требуется: провести суммирование - вывести итоговый файл, где указаны суммы по каждому материалу

N.B. Также требуется написать утилиту генерации таких csv файлов, иначе как тестировать?
Пример orders_tool.exe file1.csv file2.csv -o file3.csv
